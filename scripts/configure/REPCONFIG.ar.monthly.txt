# This file is the report configuration file for running Smkreport for
# area sources for monday, weekday, saturday, sunday reporting.

# All reports 
# Output file logical names are:
#	REPORT1: (AT.1)  No spec, no gridding, by state
#	REPORT2: (AT.2)  No spec, no gridding, by county
#	REPORT3: (AT.3)  No spec, no gridding, by SCC
#	REPORT4: (AT.4)  No spec, no gridding, by SCC and state
#	REPORT5: (AT.5)  No spec, no gridding, by SCC and hour
# 	REPORT6: (AGT.1) No spec, with gridding, by state
#	REPORT7: (AGT.2) No spec, with gridding, by county
#	REPORT8: (AGT.3) No spec, with gridding, by SCC
#       REPORT9: (ATS.1) Mass spec, no gridding, by state
#      REPORT10: (ATS.3) Mass spec, no gridding, by SCC
#      REPORT11: (ATS.4) Mass spec, no gridding, by SCC and state
